NAME

raduse - Shows the usage of the modem lines on a terminal server.  

SYNOPSIS

raduse [-w] [-o offset] terminal-server  

DESCRIPTION

raduse shows the usage of the modem lines on a terminal server. This program gets its info from the radwtmp file, which can be very slow on larger servers, and the data from the radwtmp file can't be trusted 100% either.

It is deprecated and not documented very well. Don't use it.  

OPTIONS

-w
Wide output format
-o offset
Start showing nas-port from this offset.
terminal-server
The NAS to show.

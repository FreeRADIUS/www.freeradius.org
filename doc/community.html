Community Documentation Effort

FreeRADIUS is a community-driven project. We thefore ask people to contribute documentation to the wiki. To prevent spam, we require a github or OpenID account. See the wiki for more details.

The best documentation to write is documentation that you need. If you run into a problem and solve it, please take five minutes to update the wiki. Doing so means that other people will avoid the same problem.

If no one contributes anything, the documentation will never get better.

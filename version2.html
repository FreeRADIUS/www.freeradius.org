Version 2

The "stable release" of FreeRADIUS is version 2. The goal of this version is stability. No new features will be added.

If your needs are a basic RADIUS server which runs without fail, we recommend the "stable" release. It should run for years without an issue.

The 2.2.x release series is now End Of Life. Only security fixes will be applied to 2.2.x.

It is safe to continue using version 2 for the forseeable future. If you already have version 2 running in production, we recommend that you stay with version 2 until the systems are replaced or upgraded.

We recommend that all new installations of FreeRADIUS use version 3. It has better error messages, it does more sanity checks on the configuration, the configuration is more consistent, and it is all-round better than version 2.

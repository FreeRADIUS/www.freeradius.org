Checking out from CVS

  $ cvs -d :pserver:anoncvs@cvs.freeradius.org:/source login
  CVS password: anoncvs
  $ cvs -d :pserver:anoncvs@cvs.freeradius.org:/source checkout radiusd
  $ ./configure --help
  $ ./configure ... options ...
  $ make
  $ make install
  $ radiusd -X

See the using CVS page for more details.

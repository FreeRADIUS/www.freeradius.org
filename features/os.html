Operating Systems

The server has been tested on all common operating systems. In many cases, pre-built packages are available, and are pointed to from the text below. In addition, instructions for installing it from source on many operating systems are available on the Wiki build page.

The following operating systems are supported by FreeRADIUS.

  • Linux (Ubuntu, Debian, Suse, Mandriva, Fedora Core, etc.)
  • FreeBSD
  • Mac OSX (or, in Leopard Server)
  • NetBSD
  • OpenBSD
  • DragonFlyBSD (uses NetBSD pkgsrc)
  • Solaris
  • Cygwin
  • Native Windows (in planning stages via MingW)

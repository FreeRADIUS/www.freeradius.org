Vendor Dictionaries

Almost 100 vendor dictionaries are included with the server, as outlined below. These dictionaries encompass over 4000 attributes, and over 5000 named values.

New definitions for vendors, attributes, or named values can be added in a simple textual format. This feature enables new equipment to be supported without any changes to the server source code.

We are interested in obtaining dictionaries for new vendors, or updates to the existing dictionaries. Please email updates to dictionary@freeradius.org.

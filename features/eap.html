Supported EAP Methods

FreeRADIUS is the first, and still the only, full-featured Open Source RADIUS server with support for EAP. It also supports more EAP types than most commercial products.

The following EAP methods are supported by FreeRADIUS 2.0.

  • EAP-AKA1
  • EAP-FAST1
  • EAP-GPSK (experimental)1
  • Cisco LEAP2
  • EAP-PAX1
  • EAP-PEAP3
    • MSCHAPv2 (both PEAPv0 and PEAPv1)
    • GTC (both PEAPv0 and PEAPv1)
  • EAP-PSK3
  • EAP-SAKE3
  • EAP-SIM3
  • EAP-TLS3
  • EAP-TTLS3
    • PAP
    • CHAP
    • MS-CHAP
    • MS-CHAPv2
    • EAP-MSCHAPv2
    • EAP-MD5
    • EAP-GTC
  • EAP-TNC (experimental)2

The following methods are also supported, but cannot be used with WPA or IEEE 802.1X WEP keying.

  • EAP-MD5-Challenge3
  • EAP-MSCHAPv23
  • EAP-GTC3

Notes

1   Via the "eap2" module, by using the hostapd libeap.so library. This module is experimental, and may not be ready for use in a production environment. This module is only available in FreeRADIUS versions 2.0 and later. It is not available in earlier versions of the server.

2   Via the "eap" module, using the native FreeRADIUS EAP implementation.

3   Via the "eap" and "eap2" modules. Some functionality may differ between the two implementations. See the configuration files for more details.

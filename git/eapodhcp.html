EAP over DHCP

EAP over DHCP is a protocol proposed to addess some issues seen in live networks. The code is below, and is based on the Internet Draft.

Anonymous Git Access

Anonymous git access is available for the following trees.

  • HostAP 0.6. The upstream code is in the "master" branch. The code is in the "eapodhcp" branch.
    • $ git clone git://git.freeradius.org/hostap-06.git
  • BusyBox. The upstream code is in the "master" branch. The code is in the "old-eapodhcp" branch, which is a fork of the "old" branch. The "old" branch is just a pointer to time when the eapodhcp work started. The upstream busybox code has changed significantly since then, and the EAPoDHCP patches need to be re-worked.
    • $ git clone git://git.freeradius.org/busybox.git

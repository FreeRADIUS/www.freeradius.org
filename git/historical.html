Historical Files

The git tree available here was imported from the FreeRADIUS CVS tree at cvs.freeradius.org. For speed, a local copy was used:

radiusd-cvs.freeradius.org.tar.gz (10 MB)
radiusd-cvs.freeradius.org.tar.gz.sig

The git import command was:

$ git cvsimport -kam -d ./source -C ./freeradius-server.git radiusd

If you want to reproduce the git tree yourself, you will need to create a CVSROOT directory with the appropriate information in it.

This archive is strictly for historical purposes, and should be of little interest to most people. The same information is available in the git tree.

Older Releases

Older releases are no longer actively maintained, and no help or support for these versions will be provided. These systems should be upgrade to 3.0 as soon as possible.

2.2.x Series - EOL 2015-09

The 2.2.x release series is now End Of Life as of September 2015. Users of 2.2.x and prior are encouraged to upgrade to the latest 3.0.x series release.

All 2.1 systems can be upgraded to 2.2. No configuration changes are required.

After September 2015, the 2.2 releases will no longer be actively maintained. We understand that many major Linux distributions still ship version 2.1.12 and 2.2.8. We suggest contacting your vendor, and asking them to upgrade to a supported version.

2.1.x Series - EOL 2014-12

The 2.1.x release series is now End Of Life as of December 2014. Users of 2.1.x are encouraged to upgrade to the latest 2.2.x series release, or migrate to the latest 3.0.x series release.

All 2.1 systems can be upgraded to 2.2. No configuration changes are required.

After Decemeber 2014, the 2.1 releases will no longer be actively maintained, and no help or support for these versions will be provided. We understand that many major Linux distributions still ship version 2.1.12. We suggest contacting your vendor, and asking them to upgrade to a supported version.

1.1.x Series - No longer maintained!

As of January 2008, the version 1.1.x releases are no longer actively maintained, and no help or support for these versions will be provided. Version 1.1.8 was the last release in that cycle.

There is no community support for v1.1.x. We strongly recommend that everyone using version 1.1.8 (or any earlier version) upgrade to the latest 3.0.x release as soon as possible.

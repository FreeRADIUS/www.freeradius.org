Press Releases - 2006

August 22, 2006 - Version 1.1.3 has been released.

The focus of this release is updates to libtool and libltdl, which permits the server to build more easily on many platforms.

May 26, 2006 - Version 1.1.2 has been released.

This version supports tagged VSA's for Juniper Networks equipment, configurable "cipher_list" and "check_cert_issuer" for the EAP-TLS module, and fixes a number of bugs.

May 2, 2006 - AAA for Fun and Profit

Peter Nixon, a core member of the project, is giving the keynote speech at the Free Software/Open Source Telephony Summit.

April 11, 2006 - Apple says Leopard Server will include RADIUS authentication.

The Leoppard Server Sneak Peek page says that Leopard will include RADIUS authentication for Airport Base Stations. At this weeks WWDC, they announced that the RADIUS server they are using is FreeRADIUS. This means that in the future, every MAC OSX Server installation will include FreeRADIUS.

March 20, 2006 - Version 1.1.1 has been released.

A security problem in the EAP-MSCHAPv2 module has been fixed. All users of FreeRADIUS are requested to upgrade to Version 1.1.1, wich fixes this problem. Other fixes include bug fixes for 64-bit machines, and updated handling of CoA-Request packets.

January 12, 2006 - Version 1.1.0 has been released.

After a long series of 1.0 releases, we have moved to a 1.1 version. This version includes support for load-balanced modules, a stable rlm_perl, N-tier certificates for EAP-TLS, PEAP, and EAP-TTLS, many dictionary updates, support for additional VSA formats such as Lucent and Starent, encrypted IP addresses and integers for equipment from Juniper Networks, Windows Machine Authentication support for PEAP with rlm_mschap, and many bug fixes.

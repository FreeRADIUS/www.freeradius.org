For a number of months now, the FreeRADIUS project has been collecting funds from the Amazon Associates program, mostly from the sales of the RADIUS book. Now that we have sufficient funds collected, we would like to announce the awards program.

How it works

At the end of every month, we will take the list of people who've contributed to the server for that month, and pick one to be the contributor of the month. They would get their choice of about $50 towards buying a book of their choice, (from Amazon, using the FreeRADIUS associates ID). The FreeRADIUS Project will take care of ordering the book, paying for it, and having Amazon deliver it to the award winner.

The winners name, description of their contribution, and some more information will be added below, to the Award Winners list for posterity.

We expect that the regular developers don't need much encouragement, so the chosen contributor would generally be someone outside of the regular development team. This practice encourages others to contribute, who may not otherwise have the incentive.

Every month, a goal for that month will be announced. The goal will be something which will move the development and use of the server forward. It may be more documentation. bug fixes, added features, or almost anything else. The winner does not have to contribute to that goal in order to receive the award of the month, but the goal does help to focus efforts.

Goal of the Month

For May 2003, the goal is more man pages. The documentation for the modules (rlm_*) is currently in various forms in the doc/ directory. We need to have these files reformatted in a standard manner, updated, and to have more documentation added. This is where you come in.

The steps to acheiving the goal are as follows:

  • Download txt2man, which will be used to generate man pages from plain-text files.
  • Pick a file (rlm_*) from the doc/ directory, and email the freeradius-users list, saying which file you're working on.
  • Edit the file to follow the format of the txt2man program, and run txt2man on it, to be sure that a good man page is generated.
  • If you have any questions, email the list.
  • When you've finished updating the file, email it to the list. If you are an award winner for that month, you will be contacted.

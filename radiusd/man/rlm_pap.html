NAME

rlm_pap - FreeRADIUS Module  

DESCRIPTION

The rlm_pap module provides PAP functionality.

PAP authentication works with passwords stored in clear-text format or in an encrypted ( via crypt() ) format.

The configuration item(s):

encryption_scheme
The method used to encrypt the password. Valid values are:
       clear

       crypt (DEFAULT)

       md5

       sha1

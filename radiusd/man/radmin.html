NAME

radmin - FreeRADIUS Administration tool  

SYNOPSIS

radmin [path]  

DESCRIPTION

Tool that connects to the control socket of the FreeRADIUS server, and gives a command-liner interface to the server.

At this time, only a few commands are supported. Please type "help" at the command prompt for detailed information about the supported commands.  

WARNING

This tool is experimental and should not be used in production environments. Changes may be made at any time to the commands accepted by the server, and/or to the resulting output.

       rlm_chap - FreeRADIUS Module

DDEESSCCRRIIPPTTIIOONN
       The _r_l_m___c_h_a_p module provides CHAP functionality.

       This  module  validates  a user with CHAP authentication.  If called in
       Authorize, it will look  for  CHAP-Password  attribute  in  the  Acess-
       Request and adds an Auth-Type attribute set to CHAP if the Config-Items
       list unless Auth-Type has already set.

       CHAP authentication requires access to the clear-text password for  the
       user.   Standard  Unix system authentication or passwords encrypted via
       crypt() are not compatible with CHAP.

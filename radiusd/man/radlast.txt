DDEESSCCRRIIPPTTIIOONN

       the local system.  rraaddllaasstt is a frontend for the systems llaasstt command -
       it  just  calls  llaasstt  with  the  _-_f _p_a_t_h___t_o___r_a_d_w_t_m_p___f_i_l_e argument, and
       passes all options on the command line to the system llaasstt command.
